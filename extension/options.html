Master

The master secret contains of

  • the required password you will be asked frequently for
  • and an optional username

The master password is either Saved in the browser (local storage) for 14 days or Set in memory volatile until the end of the browser session or until it is overwritten with en empty value.




The optional username is used as an additional parameter to derive passwords, even if you use the same master password like another user, different site passwords will be generated.
